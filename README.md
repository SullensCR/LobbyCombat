## **LobbyCombat**
a plugin I vibecoded for my server because the other ones suck lol

Imagine fusing a simple FFA (Free for all) gamemode with your Lobby/Hub
...that's the point of this, but more customizable and with many features I haven't seen on other plugins

*This plugin is/was made for Spigot on 1.8.8 running PandaSpigot and Java 21*

Everything is customizable (and if it isn't, let me know :3)
### Features:
- Kit system - *build your own kits*
-> Includes enchantments support and dyed armor **read below**

- Kill Streaks - *as the name suggests*
-> Announce when a player gets big kill streak of *X amount of kills* and when they lose it
- "The Coward Logic"  - *punish cowards who disconnect from the server mid fight*
-> If Y fights Z but Z disconnects in the middle of the fight Y gets the kill automatically and Z a death
 ---> Includes alerts for both players and staff
 - Anti-Interrupt - *Let players decide if they want to get 5v1d... I got this idea from Zeqa Network lol*
 -> By default its off, but the idea is to prevent players from fighting other players that are already fighting
 includes a timer to consider the player "no longer fighting" and a command to toggle it on or off, from Duels to a Free For All
- Customizable leather armor colors - *Oh look! more visual stuff to paywall or grind haha...*
--> Inludes animated "rainbow" and "pastel" colors for leather armor and a setting to change its speed
aswell as the ability to set custom HEX colors with permissions to use and a command to switch them out

  *Imagine paywalling orange for a rank or green behind an advancement...*
- MySQL support - *sync them out between instances, useful for network owners with multiple backends running this plugin lol*
- Anti-Kill Farming - *as the name suggests, a small heuristic to figure out if someone is botting their way to the leaderboards, stats and K/D ratio*
- Placeholders for **PlaceholderAPI** - *pair this with the plugin Topper and a visible leaderboard and you got engagement from players trying to get to the top*
-> Kill, death, assists count, Kill/Death ratio, current and longest winstreak and "status" placeholders (useful for custom stuff with conditions and such, like with ConditionalEvents

As I said earlier, everything or.. at least a big chunk of the plugin is customizable, from messages to permissions and other settings
